DESCRIPCIÓ

Crea els fitxers de documentació tècnica extraient els comentaris de cada fitxer de codi de l'aplicació. Els comentaris que es vulguin publicar han d'incloure un tancament d'exclamació despulles del caràcter de comentari '%' per diferenciar-los dels comentaris informàtics del codi que no han de ser publicats.

PARÀMETRES D'ENTRADA

  • path_to_file: [string] - Ruta al fitxer del qual es vol crear la documentació

VALORS RETORNATS

  • No torna cap valor
